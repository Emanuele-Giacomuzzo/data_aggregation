# Trophic position

## Author
Wei-chung Liu

## What it does
Here is the programe for calculating trophic position (as in Levin 1980) in R. 

## Input
The first line of the code tells R where your input file is.
The input should be a textfile with the following features: the first coloumn is the prey, the second coloumn is the predator and the third coloumn is the interaction strength. 

E.g: fish	bird	1

## Output
The result is recorded in a vector called tp.
